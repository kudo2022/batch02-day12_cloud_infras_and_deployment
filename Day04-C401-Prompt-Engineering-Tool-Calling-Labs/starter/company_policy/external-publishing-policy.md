## Publishing approval

- Drafts can be prepared without approval.
- Posting to external channels requires explicit user confirmation in the current conversation.
- Publishing customer-sensitive, financial, legal, medical, or unreleased product information requires human review.

## Telegram channel

- The Telegram bot can send messages only after explicit confirmation.
- If the user says "đăng", "gửi", or "post" without confirming the exact text, ask for yes/no confirmation first.
- Keep a transcript showing the confirmation before sending.

## Public claims

- Public-facing posts must avoid overclaiming model capability, benchmark results, or product availability.
- Include source links for factual claims when possible.
